# Debugging Matchmaker

## Monitor

```
bolt monitor
```

## Subscribe to NATS messages

```
bolt deploy forward staging +nats -p 8085
export NATS_URL=127.0.0.1:8085
export NATS_USER=admin
// or
export NATS_USER=bolt
export NATS_PASSWORD=password
```
